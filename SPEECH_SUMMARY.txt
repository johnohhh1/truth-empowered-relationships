1. VOICE INPUT BUTTON 🎤
   Location: Top-right of text input in Translator
   
   ┌─────────────────────────────────────────┐
   │ What do you want to say?                │
   │                                    [🎤] │
   │ ┌───────────────────────────────────┐   │
   │ │ Type or speak your message...     │   │
   │ │                                   │   │
   │ │                                   │   │
   │ └───────────────────────────────────┘   │
   └─────────────────────────────────────────┘
   
   How it works:
   ① Click 🎤 → Red pulsing = recording
   ② Speak your message
   ③ Click 🎤 again → Text appears!

2. READ ALOUD BUTTONS 🔊
   Location: On each translation result card
   
   ┌─────────────────────────────────────────┐
   │ Noticing (Inner)                  [🔊]  │
   │ "I notice my chest feels tight..."      │
   └─────────────────────────────────────────┘
   
   ┌─────────────────────────────────────────┐
   │ Words (Outer)                     [🔊]  │
   │ "When you said you'd be home by 6..."   │
   └─────────────────────────────────────────┘
   
   How it works:
   ① Click 🔊 → Hear it spoken
   ② Icon pulses while speaking
   ③ Click again → Stop

3. NEW API ENDPOINTS
   ✅ /api/transcribe - Speech-to-text (Whisper)
   ✅ /api/text-to-speech - Text-to-speech (OpenAI TTS)
   ✅ /api/analyze-conversation - Conversation analysis

   ✅ useSpeech.ts - Reusable speech functionality
